Image Steganography Project

A secure way to hide messages inside images using encryption and LSB algorithm.

About the Project

This project allows users to encode and decode messages within images using the Least Significant Bit (LSB) algorithm. Additionally, the messages are encrypted using AES encryption to add a layer of security. Users can hide a secret message within an image and retrieve it by providing the correct passphrase.

How It Works

Encode

To encode a message, you will enter the text you wish to hide, select an image, and provide a passphrase for encryption. The message is encrypted using AES and then embedded in the image using the LSB technique.

Decode

To decode a message, you need to upload the encoded image and provide the correct passphrase. If the passphrase is correct, the hidden message will be decrypted and displayed.

How to Run the Project

Follow these steps to run the project:

  • Clone the repository: Clone the repository to your local machine by running:
    • git clone https://github.com/yourusername/your-repository-name.git
  • Open the project: Open the `ImageSteganography.html` file in your web browser to start using the tool.
  • Encode or Decode: Use the "Encode" or "Decode" tab to either encode a message in an image or extract a hidden message from an image.

Features

  • Message Encryption: The message is encrypted with a passphrase using AES encryption.
  • Image Manipulation: The secret message is embedded into the image using the Least Significant Bit (LSB) method.
  • File Handling: Users can upload and download images for encoding and decoding.
  • Security: Only users with the correct passphrase can decrypt the hidden message.
