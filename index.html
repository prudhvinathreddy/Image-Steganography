Image Steganography Project

A secure way to hide messages inside images using encryption and LSB algorithm.

About the Project

This project allows users to encode and decode messages within images using the Least Significant Bit (LSB) algorithm. Messages are also encrypted with AES for added security.

How It Works

Encode

Enter a secret message, select an image, and provide a passphrase. The message is encrypted and embedded in the image.

Decode

Upload the encoded image and provide the passphrase to reveal the hidden message.
